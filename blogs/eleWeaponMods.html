This week's patch brings Weapon Modifications to the game!

New!

  • New "Weapon Mods"
  • Apply modifications to Weapons to upgrade them
  • Each weapon can have 4 different mods, one from each mod category
  • Mod Categories: Scope, Barrel, Ammo, Other
  • Mods can give laser sights, add damage, give ammo, increase accuracy reduce cooldown and more!
  • Added Laser sights(as seen in the above line)
  • Certain weapon classes can't have certain mod types
  • Example: Lobber weapons can't have Barrel mods
  • There are also set mods; these mods are designed to work well with eachother
  • Set mods will usually follow a specific build(Boiler's -> Steamtank, Unstable ->Dynamo...)
  • Weapon mods can be set in the inventory by using the gun tab

Fixes

  • Challenge mode was always selecting the highest difficulty unlocked

Changes

  • There is now a "Guns" tab in the Inventory for selecting weapon mods
  • Inventory layout slightly changed to give more room for mods
