This week's patch brings some more Quality of Life changes to the game!

New!

  • Bug where the wrong colours would be selected when starting the Designer
  • Model now waits until clothes are loaded before appearing

Changes

  • Layout of inventory changed to give more room for element icons
  • Spacing between weapon mods increased
  • The "Select No Mod" button now has an X icon
