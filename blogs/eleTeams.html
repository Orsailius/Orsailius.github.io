This week's patch brings Teams to the game!

New!

  • Custom Team mode
  • FFA or up to 2 teams
  • Can select team composition when you create a game
  • Can now set the number of kills for victory when you create a game
  • Can now join game in a team or FFA if available
  • Ally indicators over teammates' head
  • 3 new AI Opponents
  • Dynamo AI uses lightning and slow skills
  • Nightmare AI use dark and fire skills
  • Steamtank AI uses fire, ice and tank skills
  • Different opponents are selectable when creating a game

Fixes

  • Dynamo's lightning no longer sticks to you forever
  • The effect from the Polarity skill now has an icon
  • The dot effect from blast burn now burns your opponent instead of you
  • Fixed hole in the 'House' map that could get you stuck in the basement
  • Fixed main staircase in the 'House' map so AI wouldn't get stuck

Changes

  • The cooldown on Volcano has been increased from a half second to 1 second
  • Blast burn's dot damage increased
  • The stun from the Terror skill now has its own icon

Known Bugs

  • If the master client leaves, the AI resets to basic AI
  • Random AI just selects Basic AI, this is a placeholder for now
